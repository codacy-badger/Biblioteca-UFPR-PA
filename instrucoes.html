1. Consulta Local

Para registrarmos um livro na consulta local é necessário fazer um login, por enquanto eu irei disponibilizar um usuário temporário e depois irei desativá-lo, até cada um estiver devidamente cadastrado.

Para começarmos, entre neste site: bibliotecaufprpa.hol.es/login.html

Você deverá preencher o campo do usuário com: temporario; e no campo senha: temporario; e clicar em “Entrar”. Ao fazer o login, você irá se deparar com essa página:

Quando ela carregar, você poderá clicar no botão “Consulta Local”, como grifado na imagem acima. Irá abrir esta página:

  • Campo vermelho: Caixa de texto onde deverá ser inserido o código do livro.
  • Campo amarelo: Botão para inserir.
  • Campo verde: Últimos códigos registrados, irá mostrar os três últimos códigos.
  • Campo preto: Botão que irá abrir uma caixa de texto para fazer o download do histórico de um mês desejado.
  • Campo azul: Botão para autenticar o aplicativo no Android para que faça a inserção dos códigos de barras diretamente nele. (Em experimentação / Ainda não disponível para o uso).

Qualquer dúvida sobre este tópico me contate registrando um erro/bug.

2. Contador de usuários

Seguindo os mesmos passos para entrar no sistema no tópico anterior (1. Consulta Local), iremos pular diretamente para o painel de controle.

Logo após, a página representada pela figura abaixo será aberta.

  • Campo laranja: Número de usuários registrados durante o turno.
  • Campo azul: Botões “+” e “-” para adicionar um usuário ou remover do contador.
  • Campo verde: Botão adicionar, ao ser pressionado, ele irá abrir um pop-up com formulário para adicionar um dia que não tenha sido registrado no sistema.
  • Campo vermelho: Irá abrir um pop-up, ao selecionar o mês e o ano desejado, irá ser carregado uma tabela com todas informações registradas sobre aquele mês.
  • Campo amarelo: Irá abrir um pop-up com um formulário para ser inserido o dia que deseja fazer a alteração, mas atenção, a forma que deve ser inserida é: ANO/MES/DIA.

Qualquer dúvida sobre este tópico me contate registrando um erro/bug.
